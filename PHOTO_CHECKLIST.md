# VersusHub — Real Photo Checklist

For the other 4 categories (Companies, Telecom, Construction, Clothing Brands), the real logo is already wired in via Hunter.io and does the job — no separate photo needed there. Footballers deliberately use initials, not photos (see README). This checklist covers the 42 items where a real subject photo genuinely adds something.

## How to use this

1. Click a search link below — it opens Wikimedia Commons already searching for that exact item.
2. Pick the photo that actually matches (right model/year/dish/building).
3. Click through to the file page, then click **"Original file"** (or right-click the
   image → "Copy image address") to get the real, direct image URL.
4. On the same file page, note the **Author** and **License** — copy both into
   VersusHub's `photo_credit` field for that item (e.g. `Photo: Jane Doe / CC BY-SA 4.0`).
   This is required by most Commons licenses, not optional.
5. Paste the image URL into `image_url` and the credit into `photo_credit` in
   `/admin/` for that item.

If a search comes up empty or with nothing that matches well, that item is flagged
below — leave the placeholder or source a photo elsewhere (see README for other options).
